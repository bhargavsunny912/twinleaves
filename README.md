# 🛒 Product Management Dashboard (React + MUI DataGrid)

A responsive Product Management Dashboard built using **React**, **MUI DataGrid**, and **Tailwind CSS**.  
It supports searching, filtering, sorting, pagination, and detailed product view.

---

## 🚀 Features

### 📋 Product Table
- Display products using **MUI DataGrid**
- Server-side ready pagination
- Responsive layout for all devices
- Clickable rows to view product details

### 🔍 Search
- Debounced search (500ms)
- Case-insensitive filtering
- Real-time updates

### 🏷️ Filters
- Filter products by category (Fruits, Vegetables)
- Combine filters with search

### 🔃 Sorting
- Sort by price:
  - Low → High
  - High → Low

### 📄 Product Details Page
- Detailed product view
- Image, category, price, description
- Quantity selector
- Dynamic price calculation
- Back navigation

### 📱 Responsive Design
- Fully responsive (Mobile, Tablet, Desktop)
- Adaptive layouts using Tailwind CSS
- Flexible DataGrid height handling

### ⚡ Fallback Data
- Uses dummy data if API fails
- Ensures app never breaks

---

## 🧰 Tech Stack

- **React (Vite)**
- **MUI (Material UI)**
- **MUI DataGrid**
- **Tailwind CSS**
- **Axios**
- **React Router DOM**
